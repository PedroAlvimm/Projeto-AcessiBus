# 🚍 AcessiBus

Aplicativo mobile focado em acessibilidade no transporte público.

## 📖 Sobre o projeto

O AcessiBus é um aplicativo desenvolvido para melhorar a mobilidade urbana,
principalmente para:

- Idosos
- Pessoas com deficiência
- Pessoas com mobilidade reduzida

O objetivo é fornecer informações em tempo real sobre acessibilidade,
segurança e condições do transporte público.

---

## 🎯 Problema

Atualmente, usuários enfrentam:

- Falta de informação sobre acessibilidade
- Infraestrutura precária
- Falta de comunicação com órgãos públicos

---

## 💡 Solução

O AcessiBus permite:

- 📍 Localização em tempo real
- ♿ Rotas acessíveis
- 🚨 Denúncias de problemas
- 🔔 Alertas e notificações
- 🚌 Informações sobre ônibus adaptados
